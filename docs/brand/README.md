# Base de Conhecimento da Marca Bubbles

Fonte de verdade sobre a marca, produtos e comunicação. Usada como referência para
escrever copy, criar landing pages e manter consistência entre todas as LPs.

> Estes documentos NÃO são carregados automaticamente no contexto (são grandes).
> Leia o arquivo relevante sob demanda, conforme a tarefa.

## Mapa dos arquivos

| Arquivo | Quando ler |
|---|---|
| [marca.md](marca.md) | Entender quem é a Bubbles: posicionamento, valores, público, diferenciais, concorrentes, identidade visual |
| [tom-de-voz.md](tom-de-voz.md) | Antes de escrever qualquer copy: tom, termos preferidos/proibidos, conformidade legal, taglines |
| [personas.md](personas.md) | Definir o ângulo de uma LP ou anúncio para um perfil de cliente |
| [produtos.md](produtos.md) | Construir uma LP de produto: catálogo completo por linha (PRO, Essential, Xperience, Aurabian, Collora, Sensorial) |
| [operacao-banho.md](operacao-banho.md) | Falar de diluição, rendimento, custo por banho ou passo a passo do banho |

## Regra rápida de uso
Antes de escrever copy ou criar uma LP de produto:
1. Ler `tom-de-voz.md` (sempre).
2. Ler a seção do produto em `produtos.md`.
3. Se a LP tiver foco em economia/rendimento, ler `operacao-banho.md`.
4. Conferir a persona-alvo em `personas.md`.
